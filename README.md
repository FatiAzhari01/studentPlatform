# Plateforme Intelligente de Suivi des Lauréats

## Structure
- `frontend/` — React + Vite
- `backend/`  — Plain Node.js + MySQL

## Getting Started

### Backend
```bash
cd backend
cp .env.example .env   # fill in your DB credentials
npm install
node server.js
```

### Frontend
```bash
cd frontend
cp .env.example .env
npm install
npm run dev
```
